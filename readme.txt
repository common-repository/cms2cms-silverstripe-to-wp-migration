Plugin for automated data migration from SilverStripe to WordPress. Convert SilverStripe to WordPress easily and fast

== Description ==

If you wish to migrate your current SilverStripe to WordPress – CMS2CMS Migration Plugin can help to convert all data totally automatedly in a few easy-to-do steps. No coding experience or special skills are required to master and operate the tool properly.

= HOW THIS PLUGIN HELPS: =
- CMS2CMS service is favoured by first-time users because it expects no previous programming skills to work with;
- Speedy Performance – due to the automated process, a full-scale CMS2CMS migration takes very little time: from 15 min to a couple of hours;
- Free Demo Available – CMS2CMS gives its users a possibility to view a website migration process in action and to estimate the results;
- Data Mapping - lets you choose what content items and where to move them to WordPress;
- Migration Price - calculated automatedly and depends on the data you’re about to migrate to WordPress.


= WHAT WILL BE MOVED FROM SILVERSTRIPE TO WORDPRESS? =
* Pages along with the content included there;
* Attachments (images and videos);
* Users and their roles;
* Comments, comment author and comment creation date;
* Images included in the content body.

= COMPATIBLE WITH: WORDPRESS 4.1.X – 5.0.X. =
- Our support team is available live 24/7.
- For more customized migration type, CMS2CMS offers data migration service packages to order.


== Installation ==

1. Download the plugin zip file.
2. Go to the Admin Panel -> Plugins -> Add New -> Upload Plugin.
3. Choose the downloaded file in zip format and click the “Install Now” button.
4. Go to Admin -> Plugins, activate the “CMS2CMS: Automated SilverStripe To WordPress Migration Plugin”.
5. Register an account or sign in to the existing one.
6. Provide the URL address of your Existing website.
7. Download the Connection Bridge and place it into the Existing website root folder (where index.php file is located). You can find a detailed instruction here.
8. Go back to the plugin and click “Verify Connection”. You’ll be redirected to the CMS2CMS website to proceed with your migration.
